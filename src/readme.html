Rubik JS

Introdução

Projeto realizado em Vanilla Javascript a fim de estudo, com a ideia de criar um aplicativo web que forneça um cubo de Rubik para o usuário interagir e montar, oferencendo dicas de movimento baseados no algoritimo do método de Fridrich (CFOP)
